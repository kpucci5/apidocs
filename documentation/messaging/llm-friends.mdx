Large Language Model channels created by Personal AI

LLM Friends are free channels made by Personal AI for you to join and prompt an integrated LLM of your choice. Each LLM contains its own lounge and is integrated in the persona running that lounge. All responses you receive in these lounges come directly from the LLM that is running the corresponding channel.

---
title: 'Upload URL'
openapi: 'POST /upload-url'
---

Upload a URL to your Memory. Supports various web content including YouTube videos, LinkedIn profiles, news articles, and blogs.

### Body

<ParamField body="Url" type="string" required>
  URL to upload (includes public Youtube Urls, LinkedIn Profiles, News articles, blogs, etc)
</ParamField>

<ParamField body="Title" type="string">
  Title of the uploaded document
</ParamField>

<ParamField body="StartTime" type="string">
  ISO timestamp string indicating start time - utc
  Example: "2023-07-09T10:00:00.000Z"
</ParamField>

<ParamField body="EndTime" type="string">
  ISO timestamp string indicating end time - utc
  Example: "2023-07-09T10:00:00.000Z"
</ParamField>

<ParamField body="DomainName" type="string">
  Domain name of AI Persona to upload to
  Example: "justintan-work"
</ParamField>

<ParamField body="Tags" type="string">
  Comma delimited list of tags for context and reference
  Example: "linkedin,profile"
</ParamField>

<ParamField body="is_stack" type="boolean" default="true">
  Boolean to specify whether to add the document to memory. If set to false, upload will show up as draft in Documents.
</ParamField>

### Response

<ResponseField name="message" type="string">
  Confirmation message that URL is accepted and processing
</ResponseField>

<RequestExample>
```bash
curl --location 'https://api.personal.ai/v1/upload-url' \
--header 'Content-Type: application/json' \
--header 'x-api-key: {{user_api_key}}' \
--data '{
    "Url": "https://personal.ai",
    "Title": "Personal AI",
    "DomainName": "ai",
    "is_stack": false
}'

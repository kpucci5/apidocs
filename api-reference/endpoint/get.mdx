---
title: 'Upload Document'
openapi: 'POST /upload-text'
---

The Upload Document endpoint allows you to add text content to your memory stack. The text will be processed and appear in your memory stack timeline.

### Use Cases
- Upload articles or documents
- Store personal notes
- Save important text content

### Tips
- Ensure text is clean and well-formatted
- Include relevant tags for better organization
- Set is_stack to false if you want to save as draft

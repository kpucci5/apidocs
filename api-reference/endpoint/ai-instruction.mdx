# AI Instruction

> Send instructions to your AI for specific actions

## Overview
API endpoint for sending specific instructions to your AI. This endpoint allows you to interact with different AI models and execute specific commands.

## API Reference

```yaml POST /v1/instruction
paths:
  path: /v1/instruction
  method: post
  servers:
    - url: https://api.personal.ai
  request:
    security:
      - title: apiKeyAuth
        parameters:
          query: {}
          header:
            x-api-key:
              type: apiKey
              in: header
              name: x-api-key
              description: Your Personal AI API key
          cookie: {}
    parameters:
      path: {}
      query:
        cmd:
          schema:
            - type: string
              required: true
              enum: [chatgpt, search]
              description: Specifies which AI model to use for processing
      header:
        Content-Type:
          schema:
            - type: string
              required: true
              description: Must be set to application/json
      cookie: {}
    body:
      application/json:
        schema:
          type: object
          required:
            - Text
            - DomainName
          properties:
            Text:
              type: string
              description: Text instruction to send to your AI
              example: who is Einstein
            Context:
              type: string
              description: Additional context for the instruction
              example: Respond in 2 sentences or less.
            DomainName:
              type: string
              description: The domain part of the AI profile url
              example: pai-developer
            UserName:
              type: string
              description: Name of the user sending the request
              example: Leila
            SessionId:
              type: string
              description: Session identifier for maintaining conversation context
              example: d5e9f208-937c-4a9c-92bf-5fb90ce68ff6
            SourceName:
              type: string
              description: Source application name
              example: slack
            is_message:
              type: boolean
              description: Flag to indicate if the instruction should be treated as a message
              example: true
        examples:
          example:
            value:
              Text: who is Einstein
              Context: Respond in two sentences or less.
              DomainName: pai-developer
              UserName: Leila
              SessionId: d5e9f208-937c-4a9c-92bf-5fb90ce68ff6
              SourceName: slack
              is_message: true
  response:
    '200':
      application/json:
        schema:
          type: object
          properties:
            ai_message:
              type: string
              description: The AI's response to the instruction
            ai_score:
              type: number
              description: Confidence score of the AI's response
            ai_name:
              type: string
              description: The name given to the AI persona
            SessionId:
              type: string
              description: The Id of the conversation that was passed or automatically generated
            source_app:
              type: string
              description: The source from which the request was initiated from as specified in the original API request
        examples:
          example:
            value:
              ai_message: Einstein was a renowned physicist who developed the theory of relativity. He is considered one of the most influential scientists of the 20th century.
              ai_score: 95.5
              ai_name: AI Name
              SessionId: d5e9f208-937c-4a9c-92bf-5fb90ce68ff6
              source_app: api
    '401':
      application/json:
        schema:
          type: object
          properties:
            detail:
              type: string
        examples:
          error:
            value:
              detail: AI instruction api unauthorized.
        description: Authorization Failure - API key is invalid
    '503':
      application/json:
        description: Processing Failure - Instruction could not be processed
  deprecated: false
  type: path
components:
  schemas: {}
````

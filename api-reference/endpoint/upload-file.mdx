---
title: 'Upload File'
description: 'Upload a File to your Memory'
api: 'POST https://api.personal.ai/v1/upload-file'
---

Upload a file to your Memory. The request should be sent as `multipart/form-data`.

### Headers

<ParamField header="x-api-key" type="string" required>
  Your unique API key (found in App under Settings)
</ParamField>

### Form Parameters

<ParamField body="file" type="file" required>
  The file to be uploaded
</ParamField>

<ParamField body="message_request" type="string" required>
  A JSON-encoded string containing additional request parameters
  ```json Example
  {
    "DomainName": "ai",
    "Title": "Uploaded Files",
    "Tags": "document,file",
    "is_stack": true
  }
  ```
</ParamField>

### Message Request Parameters

<ParamField body="Title" type="string">
  Title of the uploaded document
</ParamField>

<ParamField body="StartTime" type="string">
  ISO timestamp string indicating start time - utc
</ParamField>

<ParamField body="EndTime" type="string">
  ISO timestamp string indicating end time - utc
</ParamField>

<ParamField body="DomainName" type="string">
  Domain name of AI Persona to upload to
</ParamField>

<ParamField body="Tags" type="string">
  Comma delimited list of tags
</ParamField>

<ParamField body="is_stack" type="boolean">
  Whether to add the document to memory. Defaults to true.
</ParamField>

### Response

<ResponseField name="message" type="string">
  Confirmation message of file processing
</ResponseField>

<RequestExample>
```bash cURL
curl --location 'https://api.personal.ai/v1/upload-file' \
--header 'x-api-key: {{user-api-key}}' \
--form 'message_request="{\"DomainName\": \"ai\", \"Title\": \"Uploaded Files\"}"' \
--form 'file=@"/path/to/your/file.pdf"'
```
</RequestExample>

<ResponseExample>
```json Response
{
    "message": "file accepted and processing"
}
```
</ResponseExample>
